# Ladon Frontend

## Getting started

### Develop

```bash
npm i
npx nx serve ladon-core
```

or

```bash
npm run ladon-dev
```

### Format

```bash
npx prettier --write <your-path>
```

### Build

```bash
npm i
npx nx build ladon-core
```
